# Gérer les états de chargement et les erreurs

En Svelte, on utiliserait peut-être des blocs {#await}. En Vue, on utilise la réactivité simple avec des variables isLoading et error. Nous l'avons déjà esquissé dans la grille, mais appliquons-le globalement avec un intercepteur Axios pour gérer les erreurs de token périmé (401).

Modifier le fichier `src/services/api.js` :

```js
import axios from 'axios';
import { useAuthStore } from '../stores/auth';
import router from '../router';

const api = axios.create({
  baseURL: import.meta.env.VITE_API_URL || 'http://localhost:3000/api',
});

// ... l'intercepteur de requête que tu as déjà ...

// NOUVEAU : Intercepteur de réponse pour gérer les erreurs globales
api.interceptors.response.use(
  (response) => response,
  (error) => {
    const authStore = useAuthStore();

    // Si l'API renvoie 401 (Token périmé ou invalide)
    if (error.response && error.response.status === 401) {
      authStore.logout();     // Déconnecter l'utilisateur, on vide le store
      router.push("/login");  // Rediriger vers la page de connexion
    }

    return Promise.reject(error); // Propager l'erreur pour un traitement local si besoin
  }
);

export default api;
```
