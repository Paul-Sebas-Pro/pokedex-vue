# Initialisation du projet avec Vite

Pour initialiser un projet Vue 3 avec Vite, on utilise la commande suivante dans le terminal :

```bash
npm create vite@latest
```
Donner un nom au projet, ici "client", car je souhaite rattacher ce projet à un backend plus tard.
Ensuite, sélection des fonctionnalités :
- Pinia pour la gestion d'état

Une fois terminé, lancer les commandes suivantes pour installer les dépendances et démarrer le serveur de développement :

```bash
cd client
npm install
npm run dev
```
Le projet Vue 3 avec Vite est maintenant prêt à être développé !
