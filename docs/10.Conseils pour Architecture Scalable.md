# Conseils

- **Composants atomiques** : Si la `HomeView` devient trop grosse, sortir la barre de recherche ou les filtres dans des petits composants dans `src/components/`.

- **Validation Joi (Front)** : On peux utiliser la même logique de validation que le backend avec des bibliothèques comme `Vee-Validate` ou simplement vérifier les champs avant l'envoi.

- **Accessibilité** :  Bien utiliser les balises `<header>`, `<main>`, `<nav>` pour que Pokédex soit lisible par tous.
