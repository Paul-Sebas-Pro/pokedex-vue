# Configurer les variables d'environnement

Actuellement, l'URL d'API est écrite "en dur" (`localhost:3000`). C'est risqué car lors du déploiement, l'API sera sur une autre adresse.

Vite utilise des fichiers `.env`.

Crée un fichier nommé `.env` à la racine de ton projet (au même niveau que `package.json`) et ajoute-y la ligne suivante :

```Plaintext
VITE_API_URL=http://localhost:3000/api
```

**Important** : Dans Vite, les variables doivent impérativement commencer par VITE_.

Désormais, le fichier ``api.js`` utilisera ``import.meta.env.VITE_API_URL``. Si l'adresse d'API change, on n'aura qu'un seul fichier à modifier !
